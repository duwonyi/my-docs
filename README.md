## VSCode shortcut
- Command palette: cmb + shift + p
- Quick open: cmb + p
- Tune your settings: cmb + ,
- Errors and Warnings: shift + cmb + m, f8
- Toggle terminal: crtl + `
- Toggle sidebar: cmb + b 
- Switch Between editors: cmb + 1, cmb + 2, cmb + 3
- Move to explorer window: cmb + shift + e
